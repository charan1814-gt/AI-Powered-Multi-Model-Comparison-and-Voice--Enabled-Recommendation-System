# 🤖 Prompt Advisor Framework - Complete Reference

## How I'll Analyze Your Prompts

Whenever you give me a prompt, I'll follow this framework:

### 1. **ANALYZE**
Check for:
- ✅ **Clarity** - Is it easy to understand?
- ✅ **Specificity** - Is it detailed?
- ✅ **Bias** - Is it neutral?
- ✅ **Completeness** - Does it have all info?
- ✅ **Safety** - Is it ethical?

### 2. **EXPLAIN** 
Point out what's wrong:
- ❌ Specific issues identified
- ⚠️ Missing information
- 🚨 Safety concerns

### 3. **IMPROVE**
Provide better version:
- ✅ Recommended improved prompt
- Clear and specific language
- Neutral tone

### 4. **OFFER ALTERNATIVES**
3 prompt variations:
1. **Beginner-Friendly** - Simplified for newcomers
2. **Expert-Level** - Advanced with depth
3. **Specific & Detailed** - Highly specific with constraints

### 5. **ENSURE SAFETY**
- Neutral language only
- No bias or leading questions
- Ethical and harmless

## Real-World Examples by Category

### 📚 1️⃣ General Knowledge
```
❌ Bad Prompt
"Tell me about Trump"

⚠️ Problems
• Too vague
• No time period
• No context

✅ Recommended Prompt
"Who is Donald Trump? Summarize his background, career, and political role in a neutral manner."

⭐ Alternative Prompts
1. Explain Donald Trump's career before and after becoming U.S. President.
2. Provide a factual overview of Donald Trump without political opinions.
```

### 🎓 2️⃣ Educational / Exam-Oriented
```
❌ Bad Prompt
"Explain AI"

⚠️ Problems
• No level mentioned
• Too broad
• Missing audience

✅ Recommended Prompt
"Explain Artificial Intelligence in simple terms suitable for an undergraduate student."

⭐ Alternative Prompts
1. Explain AI with real-world examples.
2. Compare narrow AI and general AI with examples.
```

### 💻 3️⃣ Technical / Programming
```
❌ Bad Prompt
"Code not working fix it"

⚠️ Problems
• No code shared
• No error details
• Unclear language

✅ Recommended Prompt
"I am getting a null pointer exception in Java. Here is my code and error message. Explain the issue and how to fix it."

⭐ Alternative Prompts
1. Optimize this code for better performance and readability.
2. Explain why this code fails and suggest best practices.
```

### 🏛️ 4️⃣ Opinion-Sensitive Topics (Politics, Society)
```
❌ Bad Prompt
"Is Trump good or bad?"

⚠️ Problems
• Biased question
• Leads to opinionated output
• Not neutral

✅ Recommended Prompt
"List major achievements and criticisms of Donald Trump during his presidency in a neutral manner."

⭐ Alternative Prompts
1. What policies did Donald Trump introduce, and how were they received?
2. Summarize different perspectives on Donald Trump's leadership.
```

### 🏥 5️⃣ Health & Medical (High Risk)
```
❌ Bad Prompt
"I have chest pain what medicine to take"

⚠️ Problems
• Dangerous
• Needs professional help
• Could cause harm

✅ Recommended Prompt
"What are common causes of chest pain, and when should someone seek medical attention?"

⭐ Alternative Prompts
1. Explain chest pain symptoms that require emergency care.
2. What immediate steps should someone take if experiencing sudden chest pain?
```

### 🤖 6️⃣ AI Comparison (Your Project Core Feature 🔥)
```
❌ Bad Prompt
"Which AI is best?"

⚠️ Problems
• Undefined metrics
• Subjective
• No context

✅ Recommended Prompt
"Compare ChatGPT, Gemini, and Replicate AI based on accuracy, clarity, and usefulness for students."

⭐ Alternative Prompts
1. Which AI performs better for research-based questions and why?
2. Evaluate AI tools using measurable response-quality criteria.
```

## My Prompt Analysis Formula

```
For ANY prompt you give me:

1. SCORE (0-100)
   - Clarity: 25%
   - Specificity: 20%
   - Bias: 20%
   - Completeness: 20%
   - Safety: 15%

2. IDENTIFY ISSUES
   - ❌ Critical problems
   - ⚠️ Minor issues
   - ✅ Strengths

3. PROVIDE IMPROVEMENTS
   - Corrected version
   - Why it's better
   - How to use it

4. OFFER 2-3 ALTERNATIVES
   - Different angles
   - Different complexity levels
   - Different use cases
```

## Quick Reference Checklist

### For General Questions ✅
- [ ] Clear main topic?
- [ ] Specific enough?
- [ ] Neutral tone?
- [ ] Context provided?
- [ ] Expected output format?

### For Technical Questions ✅
- [ ] Problem clearly described?
- [ ] Error messages included?
- [ ] Code/context shown?
- [ ] Expected vs actual output?
- [ ] Environment details?

### For Opinion-Sensitive Topics ✅
- [ ] Neutral language?
- [ ] No leading questions?
- [ ] No biased assumptions?
- [ ] Multiple perspectives mentioned?
- [ ] Factual basis?

### For Medical/Safety Questions ✅
- [ ] Professional help needed?
- [ ] Asking for emergency contact info?
- [ ] Not asking for medical advice?
- [ ] Focused on education?
- [ ] Safe to answer?
