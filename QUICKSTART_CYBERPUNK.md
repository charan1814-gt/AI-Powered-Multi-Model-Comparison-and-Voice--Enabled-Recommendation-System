## 🎨 Customization

### Change Primary Color
```css
/* In styles.css, find :root and update: */
--primary-color: #your-color;
--border-color: #your-color;
```

### Change Secondary Color
```css
--secondary-color: #your-color;
```

### Adjust Glow Intensity
```css
/* Modify box-shadow values in any selector */
box-shadow: 0 0 20px rgba(255, 0, 110, 0.2);
/* Increase 20px or 0.2 for more glow */
```

## 🆘 Troubleshooting

### Colors Look Wrong
- Check browser cache (Ctrl+Shift+R to refresh)
- Verify styles.css is loaded (check Network tab)

### Effects Not Working
- Ensure cyberpunk-ui-utils.js is loaded
- Check browser console for errors
- Verify element exists before adding effects

### Not Dark Enough
- Check monitor brightness
- Add `background: #0a0e27` to specific sections
- Adjust color opacity in box-shadows
